#  Insurance Blockchain Project

##  Overview
This is a decentralized insurance claim management system built using React, Node.js, and Hardhat.  
It allows users to submit claims, verify them, and track transactions on a blockchain-like system.

---

##  Tech Stack
- Frontend: React.js
- Backend: Node.js, Express
- Blockchain: Hardhat
- Database: MongoDB
- Wallet: MetaMask

---

##  Features
-  User Authentication
-  MetaMask Integration
-  Submit Claims with Image Upload
-  Claim Approval System
-  Analytics Dashboard
-  Blockchain Explorer
-  Admin Panel
